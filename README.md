- # loseq-pangu-plugin

  - 此插件会自动将 Logseq 中所有的中文字和半形的英文、数字、符号之间插入空白，汉学家称这个空白字元为「盘古之白」，因为它劈开了全形字和半形字之间的混沌。

- ## 关于「盘古之白」

  > - https://liriansu.com/posts/2018-05-07-my-coding-ocd/
  > - https://eddy.lu/posts/pangu/
  > - https://blog.pigfarm.top/2022/10/11/pan-gu-zhi-bai/
